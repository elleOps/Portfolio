01 / 02
PassForge
Password Security Analyzer

A professional-grade password analysis tool built with pure HTML/CSS/JavaScript — no frameworks, no dependencies. Implements real information-theory entropy calculation (Shannon entropy), crack-time modelling across 5 distinct attacker profiles (from casual online attack to nation-state GPU clusters), and a cryptographically secure password generator using the Web Crypto API.

Demonstrates deep knowledge of how password security actually works — not just UI/UX, but the underlying mathematics that companies like Apple and Google use to evaluate password strength in production systems.

HTML5 CSS3 JavaScript Web Crypto API Shannon Entropy Security Engineering

DecisionLabs
AI-Powered Research & Decision Platform

A full React application that integrates with the Claude AI API to provide structured decision analysis and research synthesis. Users define a decision problem with parameters, options, and context — the AI returns scored evaluations, critical factors, risk analysis, and Socratic reflection prompts.

Built to demonstrate modern frontend architecture (React + Hooks + state management), real AI API integration, and the ability to design complex interfaces that make powerful technology accessible and usable.
